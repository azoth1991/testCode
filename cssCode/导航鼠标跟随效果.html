分步走：

1. 完成对导航的排版布局

    .nav {
        display: flex;
        list-style: none;
    }

    .nav>li {
        position: relative;
        padding: 3px 6px;
    }

    .nav>li::after {
        content: " ";
        position: absolute;
        bottom: 0;
        left: 0;
        width: 0;
        border-bottom: 2px solid #000;
    }

2. 实现底部横线的移动效果

Ⅰ. 宽度的从0到100%的动画

    .nav>li::after {
        transition: all 0.2s linear;
    }
    .nav>li:hover::after {
        width:100%;
    }

3. 也是关键性的一步

Ⅰ. 无缝衔接通过为元素的宽度和位置控制

Ⅱ. 当鼠标划过时宽度改变（从左向右变长或从有向左变长），位置不变，最终达到长度100%位置为0

Ⅲ. 当鼠标划出时宽度改变（从右向左变短或从左向右变短），位置不变，最终达到长度0，位置0或100%
